Роль для установки или удаления postgresql
По-умолчанию роль устанавливает postgresql версии 13, с портом по-умолчанию: 5432
Переменная "ACTION = install" чтобы установить,"ACTION = uninstall" чтобы удалить 
Чтобы добавить новые строки в файл конфигурации postgesql.conf необходимо их перечислить в словарь - "postgresql_config_str_add"
Для внесения изменений в pg_hba.conf передаем строки в переменную "ACCESS"
Рекомендую устанавливать новую версию только после удаления старой и наличия свежего дампа БД
Рестарт постгреса происходит только при внесении измениний в переменные "postgresql_config_str_add" и ACCESS
